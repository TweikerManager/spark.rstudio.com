Function Reference version 0.1.15

Connecting to Spark

Functions for installing Spark components and managing connections to Spark.

spark_install
Provides support to download and install the given Spark version

spark_connect
Connects to Spark and establishes the Spark Context

spark_disconnect
Disconnects from Spark and terminates the running application

spark_config
Defines a configuration file based on the config package and built-in defaults

spark_log
Retrieves the last n entries in the Spark log

spark_web
Opens the Spark web interface

Reading and Writing Data

Functions for reading and writing Spark DataFrames

load_df
Loads a dataframe and provides a data source compatible with dplyr

load_csv
Loads a CSV file and provides a data source compatible with dplyr

load_json
Loads a JSON file and provides a data source compatible with dplyr

load_parquet
Loads a parquet file and provides a data source compatible with dplyr

save_csv
Saves a dplyr operation result as a CSV file

save_json
Saves dplyr operation result as a JSON file

save_parquet
Saves dplyr operation result as a parquet file
