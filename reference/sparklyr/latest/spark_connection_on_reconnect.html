Provides an extension mechanism to allow package builders to support spark_connect(reconnect = TRUE)

Usage

spark_connection_on_reconnect(scon, onReconnect)

Arguments

scon
Spark connection provided by spark_connect
onReconnect
A function with signature onReconnect(scon, sconInst) that returns an updated sconInst

Description

Provides an extension mechanism to allow package builders to support spark_connect(reconnect = TRUE)
