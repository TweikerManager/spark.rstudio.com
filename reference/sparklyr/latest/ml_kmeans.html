Computes kmeans from a dplyr source

Usage

ml_kmeans(x, centers, iter.max = 10, features = dplyr::tbl_vars(x))

Arguments

x
A dplyr source.
centers
Number of centers to compute.
iter.max
Maximum number of iterations used to compute kmeans.
features
Which columns to use in the kmeans fit. Defaults to all columns within x.

Description

Computes kmeans from a dplyr source
