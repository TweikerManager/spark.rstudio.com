Logistic regression from a dplyr source

Usage

ml_logistic_regression(x, response, features, intercept = TRUE, alpha = 0, lambda = 0)

Arguments

x
A dplyr source.
response
The prediction column
features
List of columns to use as features
intercept
TRUE to fit the intercept
alpha
The elastic net mixing parameter.
lambda
The regularization penalty.

Description

Fit a logistic model using spark.lm.

Details

See https://spark.apache.org/docs/latest/ml-classification-regression.html for more information on how regression is implemented in Spark.
